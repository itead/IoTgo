IoTgo Arduino Library

IoTgo Arduino Library is an embedded microcontroller SDK with reference firmware for Arduino boards. Users can define a device object directly by using this library and related methods to connect Arduino boards to IoTgo, which hugely simplify development process and save your time. What’s more, starters can write the hardware functional code on their own easily.

For more information, please refer to IoTgo Arduino Library Github repo

Segnix & IoTgo Segnix Library

Segnix is an open-source interface library for hardware manipulation based on Linux platform and compatible with Arduino API You can develop on it by using C/C++ and Python. Segnix is dedicated to provide a fast, efficient and unified software development environment for the person like you with a strong electronic design idea.

The latest Segnix contains IoTgo Segnix Library, and you can invoke various interfaces to manipulate devices. This is suitable for the rapid development of Raspberry Pi, Banana Pi and other single board computers.

For more information, please refer to ITEAD Segnix Github repo.
